# Smart Waste Management System

This is a web application I built to make it easier for people to report waste-related issues in their area and for authorities to manage those complaints efficiently.

## What it does
- Users can sign up and log in
- Citizens can submit complaints (with images)
- Admin can view and manage all complaints
- Simple and clean interface for easy use

## Tech used
- Python (Flask) for backend
- SQLite for database
- HTML, CSS, JavaScript for frontend

## Project structure
- templates/ → all HTML pages
- static/ → CSS and uploaded images
- app.py → main application file
- requirements.txt → dependencies

## How to run it
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000
